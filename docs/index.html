「わさびや」はなぜ「わさびや」なのか

Why “wasabiya” is “wasabiya”

わさびやは

・豊かな自然と歴史が育てた、日本の雪山文化を愛しています

・雪山文化の担い手が、鮨職人や和牛農家と同じように、世界に幸せを届けていることを知っています

・雪山文化の担い手が、課題に直面するのを見ています

・鮨や和牛に添える「わさび」の様な、雪山文化の担い手の、一助となる仕組みを作ります

「四つ葉の山葵」とは

About “four-leaf wasabi”

「四つ葉の山葵」に込めた思い

  1. 課題の現場・現物・現実を全力で理解します
  2. 実現可能な解決策を全力で考えます
  3. 結果実現のために全力で行動します
  4. 信頼を築き、積み上げます

役員紹介
  • 眞野 克彦
  • 代表取締役

1970年東京都生まれ、東京とスイスで育つ。東京大学卒業後、東京三菱銀行（現：三菱UFJ銀行）、ゴールドマン・サックス、ボストン・コンサルティング、アライアンス・バーンスタイン（東京、香港）にて金融および経営コンサルティングを20年超経験。 外国の友人に「日本でスキーがしたい」と相談されたこと、合宿していたスキー場のインバウンドでの課題を知ったことをキッカケに、2015年にシステム開発・コンサルティングを手がけるJapan Ski Holidayを香港にて創業。2019年に活動拠点を日本に戻し当社創業。 大学スキー部で一流のクロスカントリー・スキーヤーを目指した日々も今は遠く、体重増加に悩む三流半の週末スキーヤー。「選手としては無理だね。でも頑張る方法は他にもあるよ」という某コーチの言葉を胸に今日も精進中。
